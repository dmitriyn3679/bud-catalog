# Mark Phase as Done

Update the Implementation Phases checklist in `.claude/CLAUDE.md`.

Mark the phase specified in $ARGUMENTS as completed (change `[ ]` to `[x]`).

Example: if user says "phase 1", update:
```
- [ ] Phase 1 — Backend: Auth
```
to:
```
- [x] Phase 1 — Backend: Auth
```

Then confirm which phase was marked done.
